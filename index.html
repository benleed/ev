To find How often we can stack off :
ProPokerTools pql 
      select count(minHVREquity(PLAYER_1,flop,0.31))  as HOW_OFTEN 
      from game='omahahi', syntax='Generic',
      PLAYER_1='2h3s4c5d',
      PLAYER_2='AA'

To find Ave Equity : 
   select avg(equity(PLAYER_1,flop))  as AVE_EQUITY 
      from game='omahahi', syntax='Generic',
           PLAYER_1='2h3s4c5d',
           PLAYER_2='AA'
      where minHVREquity(PLAYER_1,flop,0.31)


(Just substitute '0.31' with the min equity required)
